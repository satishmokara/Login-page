# Login-page
# 🚀 Secure Login System (Next.js + Vercel)

A clean, responsive, and functional login application built with **Next.js**, **React**, and **Tailwind CSS**. Designed for seamless deployment on **Vercel**.

## ✨ Features
* **Responsive UI:** Fully optimized for mobile, tablet, and desktop.
* **Serverless Backend:** Built-in API routes using Next.js.
* **Postgres Ready:** Database schema included for user management.
* **Vercel Optimized:** Ready for one-click deployment.

---

## 🛠️ Tech Stack
* **Frontend:** [Next.js](https://nextjs.org/) (React)
* **Styling:** CSS Modules / Tailwind CSS
* **Database:** [Vercel Postgres](https://vercel.com/docs/storage/vercel-postgres) (SQL)
* **Deployment:** [Vercel](https://vercel.com/)
